# Applescripts Directory

```
chrome: 
    SYNOPSIS:
        osascript chrome.scpt [string_to_search]
    DESCRIPTION:
        Searches the string passed in via google.

spotify:
    SYNOPSIS:
        osascript spotify.scpt
    DESCRIPTION:
        If spotify is running, play Stairway to Heaven by Led Zeppelin

volumemute:
    SYNOPSIS:
        osascript volumemute.scpt
    DESCRIPTION:
        Sets the volume to 0. 

volumecycle:
    SYNOPSIS:
        osascript volumecycle.scpt
    DESCRIPTION:
        Oscillates the volume up/down at a reasonable pace 5 times. 
```
