# Applescripts Directory

```
chrome:
    SYNOPSIS:
        osascript chrome.scpt
    DESCRIPTION:
        Searches the string in the file search_string.txt

spotify:
    SYNOPSIS:
        osascript spotify.scpt
    DESCRIPTION:
        If spotify is running, play the spotify track in the spotify_song.txt file. Default is "Uprising" by Muse

volumemute:
    SYNOPSIS:
        osascript volumemute.scpt
    DESCRIPTION:
        Sets the volume to 0.

volumecycle:
    SYNOPSIS:
        osascript volumecycle.scpt
    DESCRIPTION:
        Oscillates the volume up/down at a reasonable pace 5 times.
```
